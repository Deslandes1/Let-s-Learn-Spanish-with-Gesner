# Let's Learn Spanish with Gesner

Interactive Spanish learning app (Book 1, Lessons 1–20) built with Streamlit.

## Features
- 20 thematic lessons
- Conversations, vocabulary, grammar, pronunciation (text-to-speech)
- Interactive quizzes
- Password protected (demo: `20082010`)

## Run locally
```bash
pip install -r requirements.txt
streamlit run app.py
